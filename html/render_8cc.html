Detailed Description

2D render library for the XPS DVI interface as exposed in the NEST platform.

This library uses screen subdivisions (or subdivs for short). This means that multiple threads can be used to render a subdiv of the screen. All functions have a xsubdiv and ysubdiv parameter, which indicate which subdiv is intended to use. When set, a thread cannot write outside of its subdiv; all images are clipped. When either is set to -1, clipping is overruled.

Created on: May 13, 2011 Author: Berend Dekens <b.h.j.dekens@utwente.nl> Version: 2

History: 2:

  • Complete rewrite. 1.2:
  • Added support for memory consistency and fully cached external memory
  • Added rectangle fill mode with alpha transparency support 1.1:
  • Fixed a bug in the fillrect where too many pixels are drawn in the horizontal direction on non-multiples of 32 pixels in width
  • Added a raw image drawing routine and a raw pixel drawing function 1.0:
  • Feature complete for the visual spectrum analyser
