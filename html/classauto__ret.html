Detailed Description

template<typename T, T ok>
class auto_ret< T, ok >

Class to automatically verify return values, even when caller does not.

An instance of this class can be used as the return value of a function. When the function returns a value that should be handled by the calling function, such as a non-zero return value of pthread_create(), and this value is not read, the destructor will abort() the program.

Effectively, this removes the need to clutter your code with all kinds of error checking for functions that are never throwing errors, but still have a safety net when they do.
