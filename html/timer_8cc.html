Detailed Description

Low-level functions to control the local timer for interrupts.

These functions should only be used by the kernel to manage the time-slices for scheduling.

Sets the local timer to the designated number of clock cycles.

The timer will be started automatically. When the value 0 is reached, an interrupt is generated.

Parameters:
timenumber of cycles to set the counter to
repeatset to non-zero to reload and restart the timer with this interval when zero is reached
