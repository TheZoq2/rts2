Detailed Description

Tracing of process execution and phases.

An application can call TracePhaseChange() to indicate a different phase. All other kTrace*() calls should not be called from within an application. Based on the recorded trace stream, execution times can be calculated.

The size of a trace of a single processor is determined by TRACE_SIZE, which is the number of bytes that is malloc()ed on the processor heap. Traces can be converted to VCD by using the trace2vcd tool.
