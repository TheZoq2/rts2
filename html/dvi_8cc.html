Detailed Description

Low-level functions to control the DVI port.

The supported video format is 640x480 pixels, 60 Hz, 24 bit colors. Frame lines are 1024 pixels wide, but only the first 640 pixels are shown on the screen. Pixel information is stored as follows: (msb) --------rrrrrr--gggggg--bbbbbb-- (lsb).
