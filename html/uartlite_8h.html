Read a string/line to a UART described by the uartlite_t structure.

Note: if maxcount is reached before a
is received, the function returns the data up to that point. Otherwise it reads a single line (
delimited ) and returns the number of bytes read until the EOL was detected.

Parameters:
upPointer to the UART device
strPointer to a string to fill with bytes from a UART device
maxcountNumber of bytes to write into the UART device, use -1 to write the entire string (up to the null character)
Returns:
Returns the number of bytes actually written to the UART device
