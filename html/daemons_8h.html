int dBootKernels ( int  num,
int *  procs 
)

Boots the kernel on other processors.

It does the following: other processors will get the boot signal (assuming the slaveloader is running) and we will wait for the boot notification. When you want to use the wrapper functions like CreateProcess(), all other cores must be booted using this function, in order to setup the channels correctly. Do not call this function twice.

Parameters:
numthe number of processors to boot, that is the length of the procs parameter
procsan array of core IDs of processors to boot
Returns:
0 on success, EXFULL when the fifo memory is full, otherwise an errno. In case of an error, it is undecidable which processors have been booted and which have not.

void* MallocDaemon ( void *  arg)

Daemon to handle all processor heap operations.

Every process can have one outstanding malloc/calloc/realloc/mown/free request. The daemon will handle all requests in order of process id. When all requests are processed, the daemon Yield()s. This means that all heap requests are serviced in one complete scheduling period, assuming that the daemon finishes before the end of its timeslice. The heap will not be locked; locking is implicit by using this daemon.

The FIFO heap and shared heap are not managed by this daemon. The daemon should be started by calling StartMallocDaemon().

Parameters:
arga pointer to a malloc_buffer_t[MAX_PROCESSES*2], cannot be null
Returns:
(void*)errno
