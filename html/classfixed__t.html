Detailed Description

template<uint8_t I, uint8_t P, typename S = SignedType, fixed_rounding_t rounding = round_nearest_int>
class fixed_t< I, P, S, rounding >

A fixed point type, with configurable primitive type, bits after comma, and rounding policy.

The implementation is heavily optimized such that it is compiled to only a few, fast instructions. The template class has the following parameters: I is the number of bits before the comma; P is the number of bits that is used after the comma (so when having T=int, one bit is used as the sign bit, P bits are used for after the comma, and 32-1-P bits are used for the integer part); S is either SignedType or UnsignedType; rounding is the rounding policy when the type is casted to an integer value. Use round_truncate to emulate int behavior, round_half_up for efficient unsigned rounding, and round_nearest_int for the slowest, but most accurate signed rounding.

During coding, don't think about all these overloaded operators, just use the type as you would do with an int or float.
