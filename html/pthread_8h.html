Detailed Description

POSIX Pthread standard implementation.

A mutex, condition and barrier are synchronization variables. For memory consistency, they are assumed to be always shared objects, but they need not to be protected. So, when using i.e. a mutex, do not wrap the pthread_mutex_* calls in mc_entry_* and mc_exit_* calls. In contrast to any other variable, these are automatically cache line aligned and protected by themselves.
