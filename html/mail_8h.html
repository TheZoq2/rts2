Detailed Description

Mailboxes: a simple system to pass integer values around.

The concept of a mailbox is easy: it is a pointer to a memory. The writing side is only allowed to write once, the reading site can read it as many times as wanted.

A simple mailbox is just one 31-bit value (the last bit is set when the value has been written). A normal mailbox can contain multiple values. A counter indicates how many fields in the box are written. Normal mailboxes must be written in increasing index -- a write to index n means that all values at an index lower than n are valid.

These mailboxes can be used to signal between processors, such as the result of a command that has been executed. Using FIFOs for this is possible, but a FIFO has much more overhead.
