int BootKernel ( int  procid)

Boots the kernel on another processor.

When one of the wrapper functions like CreateProcess() is to be used, make sure that the other kernel is booted using BootKernel() or dBootKernels().

Nothing happens when procid is set to the current processor ID. Do not boot a processor twice.

When BootKernel() is called repeatedly for a list of processors, consider using dBootKernels() directly; that reduces the overhead significantly by skipping the already-booted check and booting all cores in parallel.

Parameters:
procidthe ID of the processor to boot
Returns:
0 on success, otherwise an errno
