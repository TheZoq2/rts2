ssize_t linfs_write ( int  fd,
const void *  buf,
size_t  cnt 
)

Writes buffer to the file descriptor.

In contrast to POSIX write(), partial writes will automatically retried until nothing is written anymore or an error occurs.

As a consequence, when linfs_write() returns with a value less than cnt, errno always indicates the error.

When fd has been open()ed with flags&O_ASYNC, the write will return immediately with cnt.

In that case, errors during the write will be silently ignored.
