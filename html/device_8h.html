Detailed Description

Declaration of functions that gives information about the hardware platform.

The values are read from the local memory that contains the .devinfo section.

unsigned int DisableDCache ( void  )

Disables data cache (when available).

Returns:
A number that indicates how slow the memory was (mainly for debugging)

When turning off the cache, all valid lines will be written to the memory. So, turning off and on the cache at any point in the program is safe. The cache will be completely invalidated when DisableDCache() returns.
